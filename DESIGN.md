# Design System

## Theme: Football Pitch

### Color Palette
- **Background**: Deep grass green (`#1a6b2e`)
- **Accents**: White (`#ffffff`) and golden yellow (`#f5c518`)
- **Text**: White for primary text, golden yellow for highlights, deep green for text on white backgrounds.

### Typography
- **Headings**: `Bebas Neue` (tall, athletic, zero-clutter)
- **Body**: `Inter` (clean and minimal)
- **Typography Rules**:
  - Hero display text must be massive and impactful.
  - Headings should use `text-wrap: balance`.

### Visual Elements
- **Decorative**: White SVG pitch lines (center circle, sidelines) as a subtle background layer.
- **Spacing**: Whitespace is the primary layout tool. Sections must breathe.
- **Components**: No complex nested cards. Simple, punchy interactive elements (e.g., 1-5 star picker, simple toggle buttons).
